# kivy-examples

## ru
### про код
Три простые программы с примерами кода на Python и фреймворке Kivy. Подробное описание можно найти в статье. 
Материал создан для Академии Selectel: selectel.ru/blog/author/alexdzhalatov

### подготовка к запуску
В репозитории есть три директории. В каждой из них, для запуска вам потребуется создать виртуальное окружение, в котором нужно установить:
- python
- kivy
- kivymd
- pyinstaller (опционально)

После подготовки окружения вы можете выполнить отладку запустив main.py или выполнить сборку проекта с помощью pyinstaller.

Three simple programs with Python and Kivy framework code examples. For an accurate description, read the article.

### prepare to launch
There are three directories in the repository. In each of them, to run, you will need to create a virtual environment in which you need to install:
- python
- kivy
- kivymd
- pyinstaller (optional)

After setting up the environment, you can debug by running main.py or build the project using pyinstaller.
